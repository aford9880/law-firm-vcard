# Юридическая компания «Правовед»

Одностраничный адаптивный сайт-визитка для юридической фирмы.

![Скриншот](screenshot.png)

## Демо-видео

## Стек

- HTML5 + CSS3 (Flexbox/Grid)
- Vanilla JS (слайдер отзывов, валидация формы)
- Jest (тесты)

## Запуск

```bash
# открыть в браузере
start index.html

# тесты
npm install
npm test
```
