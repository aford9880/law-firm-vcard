# Юридическая компания — План ⭐

## Цель
Одностраничный адаптивный сайт-визитка для юрфирмы. Запись на консультацию — главная конверсия.

## Структура
1. **Header** — логотип, навигация (якоря), CTA «Записаться»
2. **Hero** — заголовок, подзаголовок, фон, кнопка «Бесплатная консультация»
3. **Услуги** — карточки: семейное, трудовое, гражданское, налоговое право
4. **Преимущества** — опыт, гарантии, конфиденциальность, скорость
5. **Отзывы** — слайдер/карусель с цитатами
6. **Форма заявки** — имя, телефон, услуга (select), комментарий → отправка
7. **Контакты** — адрес, телефон, email, соцсети
8. **Footer** — копирайт, политика конфиденциальности

## Технологии
- HTML5 + CSS3 (Flexbox/Grid)
- Vanilla JS (форма, слайдер отзывов)
- Базовая SEO (meta-теги, Open Graph)
